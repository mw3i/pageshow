Usage

  1. Download this repo

  2. Install pandoc and python (probably have to use a unix based os) if you don’t have them

  3. edit make.py file; change the doc variable to the path of the document you want to convert

  • you can also change the heading level you want to split at (1,2,3,… -> #,##,###,…)
  1. run the make.py file in the terminal with:

python3 make.py

^ or whatever it is you use to call a python script

  1. open up the index.html file in your browser to view
  • use right/left arrow keys or mouse click to navigate
